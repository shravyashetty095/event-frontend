# Event Management System (Frontend)

This is the frontend of the Event Management System built using React.

## Features

* User login and registration
* Event listing and booking
* Responsive UI design

## Tech Stack

* React.js
* HTML, CSS, JavaScript

## Run Locally

```bash
npm install
npm start
```

## Deployment

This project is deployed using Netlify.

## Note

Backend APIs are not deployed in this version. Some features may not work without backend connection.
